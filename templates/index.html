Premise Health University - Duty Calendar

Keeping your Duty Legal

For this example, I built a Node.js + hapi.js server. I implemented a basic userstore and persistance with MongoDB. (Using MongoLab in the cloud so it can be accessed from anywhere.) I chose to use hapi.js and Angular.js, despite limited experience with either, to have this challenge double as a learning experience. It was a great deal of fun, thanks for the opportunity!

There are some basic pages for user registration, login, etc.

There is a page that allows the user to see their shifts and add new shifts. There is also an analysis page to show the user their shift violations.

Angular.js was used for page interactivity and with the 'API' on server to determine user state and to request violations. All violation logic is server side. jQuery was used lightly for the date picker on the calendar page.

Technologies Used: Node.js, hapi.js, Angular.js, Bootstrap, Joi, Boom, MongoDB, Moment.js, MongoLab, jQuery, bcrypt
